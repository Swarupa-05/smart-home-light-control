## Project Scope

This project is developed as part of my internship training. It demonstrates a Smart Home Automation System using ESP32 microcontroller.

The system uses sensors like LDR (Light Dependent Resistor) and PIR motion sensor to automatically control a light (LED). It also includes a manual/auto mode switch using a push button and displays real-time sensor data on an OLED display (SSD1306).

The project simulates real-world smart home automation where lighting is controlled based on environmental conditions and human presence.
